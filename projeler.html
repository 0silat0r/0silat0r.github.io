1. LED Yakıp Söndürme (Blink)

Gerekli Malzemeler
  • Arduino UNO R3
  • 1x Herhangi bir renk LED
  • 1x 220 Ohm Direnç
  • Breadboard ve Jumper Kablolar
Yazılacak C Kodu
// Arduino Uno Blink C Kodu void setup() { pinMode(13, OUTPUT); // 13 numaralı pini çıkış yaptık } void loop() { digitalWrite(13, HIGH); // LED'i yak delay(1000); // 1 saniye bekle digitalWrite(13, LOW); // LED'i söndür delay(1000); // 1 saniye bekle }
